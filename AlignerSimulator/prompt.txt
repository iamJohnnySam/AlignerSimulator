Create me a visual app which simulates a semiconductor wafer aligner. I want to see the data that the CCD aligner line sensor would see when the wafer is spinning on the chuck. 
This is the sensor I am using - (Keyence IG-028) keyence.com/products/sensor/positioning/ig/models/ig-028/
The sensor is aligned such that the line sensor is perpendicular to the tangent of the chuck. The value towards the chuck is 0.00mm and the furthest value from the chuck will be 28.00mm. Ideally the wafer edge will be at half the length

I want the following parameters to be editable

size of the wafer (150mm, 200mm, 300mm, ...) (The wafer size can be increments of whole numbers.)
notch starting position
Amount of noise on sensor
Amount of wafer warpage as bow or potato chip shape
Amount and size of random wafer edge chips
offset of the wafer on the chuck
chuck rotation speed
sampling speed of the sensor

I want to swap between wafers and tape frame. In the tape frame case I should see the edge of SEMI G74 frame while wafer should be able to swap between 100 - 450mm wafers.
 If tape frame is selected we dont need to worry about the wafer on the inside. We only look at the outer shape of the tape frame which is seen by the sensor.

Visually I want to be able to control the rotation of the wafer so I can see the behaviour of the sensor based on the sensor position.
Once I am happy with the parameters I want to be able to export the sensor data output for given number of rotations as a csv file.
